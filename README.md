# FastAPI AITeeth

Using FastApi to deploy an ai model for detecting teeth in a smile.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/).(use the last version of pip)

install FastAPI.

```bash
pip install fastapi
```
install uvicorn for running FastAPI.

```bash
pip install uvicorn
```

install OpenCV (cv2).

```bash
pip install opencv-python
```
install Cmake for dlib.

```bash
pip install cmake
```

install dlib.

```bash
pip install dlib
```

install requests.

```bash
pip install requests
```


## Usage
start the app

```bash
uvicorn main:app --reload
```

test the api
```bash
python test_api.py
```
